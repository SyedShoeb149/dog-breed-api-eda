# dog-breed-api-eda
Exploratory Data Analysis on hierarchical dog breed data using public REST API and Python.

🐶 Dog Breed API – Exploratory Data Analysis
📌 Project Overview

This project explores hierarchical dog breed data obtained from a public REST API.

The goal was to:

Extract JSON data from an API

Normalize nested structures into tabular format

Perform exploratory data analysis (EDA)

Visualize sub-breed distribution patterns
