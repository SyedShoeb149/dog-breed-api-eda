# dog-breed-api-eda
Exploratory Data Analysis on hierarchical dog breed data using public REST API and Python.

🐶 Dog Breed API – Exploratory Data Analysis
📌 Project Overview

This project explores hierarchical dog breed data obtained from a public REST API.

The goal was to:

Extract JSON data from an API

Normalize nested structures into tabular format

Perform exploratory data analysis (EDA)

Visualize sub-breed distribution patterns

🛠️ Tech Stack

Python

pandas

requests

matplotlib

📊 Key Analysis

Distribution of breeds with and without sub-breeds

Identification of breeds with highest subtype diversity

Frequency analysis of recurring sub-breed labels

Visualization of categorical distributions

📚 Key Learning Outcomes

Handling nested JSON data

API data extraction workflows

Data transformation using pandas

Hierarchical relationship modeling

🚀 Future Improvements

Store data in SQL database

Automate API extraction pipeline

Deploy as small dashboard
